todo
====

A ClojureScript todo app that runs in the browser.  A test project.

All of the relevant code is in `cs-todo/src-cljs/todo/core.cljs`.

Usage
-----

    $ lein deps
    $ lein cljsbuild once
    $ open resources/public/index.html
